# Overview
* Structural
* Also called Wrapper (as it wraps the original object )

# Intent
* Add functionality to existing objects dynamically
* Alternative to sub classing
* Flexible design
* Support Open Closed Principle

## When to use
* Legacy Systems
* Add functionality to controls
* Sealed classes
